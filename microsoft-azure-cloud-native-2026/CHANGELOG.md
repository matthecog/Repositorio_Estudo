## [0.3.0] — Arquitetura Cloud-Native e Laboratórios

### Adicionado
- Pasta `04-laboratorios` para centralização dos ambientes práticos
- Laboratório 07 — Construção de aplicação de aluguel de carros totalmente Cloud-Native
- Deploy de API containerizada no Azure Container Apps
- Implementação de mensageria com Azure Service Bus (filas)
- Processamento orientado a eventos com Azure Functions
- Integração com Azure SQL Database e Azure Cosmos DB
- Persistência de arquivos no Azure Storage (Blob)

### Arquitetura
- Implementação de arquitetura orientada a eventos
- Comunicação assíncrona entre serviços via filas
- Separação clara entre API, mensageria e processamento
- Aplicação de princípios de desacoplamento e escalabilidade
- Estrutura preparada para crescimento e expansão futura

### Documentação
- README estruturado no padrão oficial do repositório
- Inclusão de evidências visuais dos recursos Azure
- Organização do laboratório como item independente do desafio
- Melhoria na descrição das decisões arquiteturais
