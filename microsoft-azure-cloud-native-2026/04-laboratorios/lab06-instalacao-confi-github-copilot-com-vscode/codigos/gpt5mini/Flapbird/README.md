# Flapbird Game

Flapbird is a simple yet engaging game where you control Mario as he navigates through a series of pipes. The objective is to avoid the pipes and score points by successfully passing through them.

## Getting Started

To get started with Flapbird, clone the repository and open the `index.html` file in your web browser.

```bash
git clone <repository-url>
cd Flapbird
```

## How to Play

- Press the spacebar to make Mario jump.
- Avoid hitting the pipes.
- Try to score as many points as possible by passing through the gaps in the pipes.

## Game Features

- Simple and intuitive controls.
- Engaging graphics and sound effects.
- Increasing difficulty as the game progresses.
