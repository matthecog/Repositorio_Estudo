# Construa um Agente de Pedidos de Pizza com Microsoft Foundry e MCP 🚀  

![](/public/banner.png)

## Visão Geral do Workshop  

Nesta sessão prática, você aprenderá a construir agentes de IA inteligentes e específicos de domínio usando o **Foundry Agent Service**.  
Vamos passo a passo — desde criar um agente básico até estendê-lo com ferramentas personalizadas, dados externos e integrações ao vivo.  

Ao final do workshop, você terá construído seu próprio **Contoso PizzaBot**, um assistente de IA capaz de:  
- Seguir **prompts de sistema** personalizados  
- Usar **geração aumentada por recuperação (RAG)** para responder perguntas de documentos personalizados  
- Chamar **ferramentas personalizadas** como uma calculadora de pizza  
- Integrar com um **servidor MCP** para gerenciamento de menu e pedidos ao vivo  

## Agenda

- **Boas-vindas e Configuração**  
  Introdução, objetivos do workshop e configuração da assinatura Azure  

- **Capítulo 1: Crie Seu Primeiro Agente**  
  Faça login no Azure, instale pacotes e construa um agente GPT-4o simples  

- **Capítulo 2: Prompts de Sistema e Instruções**  
  Aprenda como prompts moldam o comportamento do agente e adicione instruções personalizadas do `instructions.txt`  

- **Capítulo 3: Adicionando Conhecimento (RAG)**  
  Fundamente seu agente em dados de lojas de pizza usando File Search e um vector store  

- **Capítulo 4: Chamada de Ferramentas**  
  Estenda seu agente com uma função calculadora de pizza personalizada e integração de toolset  

- **Capítulo 5: Integração MCP**  
  Conecte a um servidor MCP para menus de pizza ao vivo, coberturas e gerenciamento de pedidos  

- **Testes e Encerramento**  
  Teste o PizzaBot de ponta a ponta, Q&A e próximos passos  

## O Que Você Vai Precisar  

- Um navegador e acesso ao [Portal Azure](https://portal.azure.com)  
- Uma [assinatura Azure](./get-azure) fornecida ou use a sua própria
- Um [ambiente de desenvolvimento](./dev-environment) com Python 3.10+ instalado  
- Familiaridade com o básico de Python (não é necessário conhecimento profundo em IA!)  

## Objetivo do Workshop  

Ao final desta sessão de 2 horas, você saberá como:  
1. Criar e configurar um agente no Microsoft Foundry  
2. Guiar seu comportamento com **prompts de sistema**  
3. Fundamentá-lo em dados reais usando **RAG**  
4. Estender sua funcionalidade com **ferramentas personalizadas**  
5. Conectá-lo a serviços externos com **MCP**  

Você sairá com conhecimento prático e um agente **PizzaBot** funcional que pode expandir para casos de uso do mundo real.
